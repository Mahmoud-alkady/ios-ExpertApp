# ios-ExpertApp

### Overview:
IOS Swift Application using MVVM Architecture.
This is a simple design application for E-commerce using MVVM architecture and CoreData.

### MVVM

## Key Features
- MVVM Architecture
- Splash Screen
- Caching
- Core Data
- Extensions for UI
- Storyboards for UI
- IOS 13, 14 && 15
